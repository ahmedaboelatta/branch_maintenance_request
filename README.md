# Branch Maintenance Request | طلبات صيانة الفروع

## English Description
This Odoo module allows branch managers or users to create and track maintenance requests specifically for different branches. It helps in organizing the maintenance workflow and ensuring all issues are addressed promptly.
- **Compatibility:** Odoo 18.0
- **Main Features:** - Create branch-specific maintenance requests.
  - Track request status.
  - Integration with Odoo's standard maintenance flow.

## الوصف بالعربية
هذا الموديول المخصص لنظام أودو يسمح لمديري الفروع أو المستخدمين بإنشاء ومتابعة طلبات الصيانة الخاصة بالفروع المختلفة. يساعد النظام في تنظيم سير عمل الصيانة وضمان معالجة جميع المشكلات بسرعة.
- **التوافق:** أودو 18.0
- **الميزات الرئيسية:**
  - إنشاء طلبات صيانة مخصصة لكل فرع.
  - تتبع حالة الطلبات.
  - التكامل مع نظام الصيانة القياسي في أودو.

## Installation | التثبيت
1. Copy the folder `branch_maintenance_request` to your Odoo `custom-addons` directory.
2. Restart your Odoo server.
3. Activate **Developer Mode**.
4. Go to **Apps** > **Update Apps List**.
5. Search for `branch_maintenance_request` and click **Install**.

1. قم بنسخ مجلد `branch_maintenance_request` إلى مسار الإضافات الخاص بك `custom-addons`.
2. أعد تشغيل سيرفر أودو.
3. فعل **وضع المطور**.
4. اذهب إلى **التطبيقات** > **تحديث قائمة التطبيقات**.
5. ابحث عن `branch_maintenance_request` واضغط **تثبيت**.
